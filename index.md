# CMPN202 – Operating Systems Coursework

## Project Overview
This project documents the configuration, security hardening, monitoring, and evaluation of a Linux server operating system.
All administration is performed remotely using SSH from a separate workstation.

## System Summary
- Server OS: Ubuntu Server (Headless)
- Access Method: SSH only
- Security: Firewall, key-based SSH, fail2ban, AppArmor
- Monitoring: Custom shell scripts and CLI tools
